# ETL Sales Project

## Description
ETL pipeline that reads CSV sales data, cleans invalid rows,
applies business rules, aggregates per user, and exports JSON.

## Technologies
- Python
- csv / json

## Steps
1. Read CSV
2. Filter paid orders
3. Handle invalid data
4. Transform & aggregate
5. Export clean JSON
6. Log errors

## Output
- sales_clean.json
- errors.log
